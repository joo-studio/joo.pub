    彩信接口接口说明
    接口地址
    地址 描述
    http://cf.51welink.com/submitdata/mmswebinterface.asmx 发送彩信
    调用SubmitMms方法
    类型 命名 描述
    接口定义 SubmitMms(string UserID,string Password,string CorpID,string Prdid,string DstID,string MmsSubject,string MmsBuffer) 提交短信
    参数 UserID 提交账户
    参数 Password 提交账户的密码
    参数 CorpID 企业代码（扩展号，不确定请赋值空）
    参数 Prdid 产品编号
    参数 DstID 接收号码间用英文半角逗号“,”隔开，触发产品一次只能提交一个,其他产品一次不能超过10万个号码
    参数 MmsSubject 彩信标题
    参数 MmsBuffer 彩信tms格式内容
    调用SubmitMmsWithEms方法
    类型 命名 描述
    接口定义 SubmitMmsWithEms(string UserID,string Password,string CorpID,string Prdid,string DstID,string MmsSubject,string EmsBuffer) 提交短信
    参数 UserID 提交账户
    参数 Password 提交账户的密码
    参数 CorpID 企业代码（扩展号，不确定请赋值空）
    参数 Prdid 产品编号
    参数 DstID 接收号码间用英文半角逗号“,”隔开，触发产品一次只能提交一个,其他产品一次不能超过10万个号码
    参数 MmsSubject 彩信标题
    参数 EmsBuffer 彩信ems格式内容
    返回结果
    返回值CSubmitState结构说明
    名称 类型 描述
    State int 提交结果值,返回值为0时，表示提交成功
    MsgID string 短信批次号
    MsgState int 保留，扩展用
    Reserve String 是
    返回值枚举
    State MsgState
    0 提交成功
    -99 异常
    101 提交参数不可为空，或参数格式错误
    102 发送时间格式不正确,正确格式为yyyy-MM-ddHH:mm:ss
    103 调用速度超过限制，限制作用在接口上的某一个函数上，如果短时间内对接口上的某个函数调用频率太高(大于100次/秒)，将返回此错误编号
    201 Ems格式转换错误
    202 Tms内容异常
    203 Mix格式彩信增加Smil文件错误
    1007 错误的信息类型：XXX
    1008 超过最大并发提交用户数XXX
    1010 信息内容为空或超过最大信息字节长度XXX
    1011 超过最大企业号码长度:XXX,或企业号码不包含:XXX
    1012 企业号码未包含:XXX
    1013 账户未启用或鉴权失败,User=XXX,Pwd=XXX
    1014 账户提交方式不正确或Ip受限:Mode=XXX,Ip=XXX
    1015 提交速度受限:XXX条/秒，指用户的提交速度大于规定的单个用户的最大提交速度(默认5000次/秒)
    1016 产品不存在或未开启:XXX
    1017 提交信息类型与产品信息类型不符合
    1018 超过产品发送时段:Begin=XXX,End=XXX,Curr=XXX
    1019 提交彩信必须有标题XXX
    1020 提交短信不可超过XXX个字
    1021 提交彩信不可超过XXXK
    1022 消息拆分失败，指长短信在进行拆分后的信息条数等于0或者大于10
    1023 无效计费条数
    1024 信息内容包含关键字:XXX
    1025 Account:XXX计费失败
    1026 提交至调度中心失败
    1027 提交成功，信息保存失败
    1028 账户%s无对应的产品%d
    1029 扩展产品%d不可提交多个号码
    1031 提交时间[%s]+31天>定时发送时间[%s]>提交时间[%s] 规则不成立
    1032 无签名或者签名规则不正确，具体规则咨询客户经理,譬如“你好,欢迎光临【某某公司】”
    1033 产品%d的正则签名%s配置有误
    1034 内容中不能包含“|”。“|”系统中作为内容拆分分隔符使用
    1035 强制签名的产品%d,签名格式不正确
    1036 企业代码不正确
    1037 未成功加载账户强制签名报备模块
    1038 强制签名的产品%d,签名%s未报备
    1039 未成功加载白名单模块
    1040 消息提交成功，但消息编号生产失败
    1041 未成功加载账户内容模板模块
    1042 内容不符合模板
    1043 未成功加载账户安全登录模块
    2001 余额不足

    注意：提交结果的接收只需判断State是否等于0就可以，State等于0就表示信息已提交成功
