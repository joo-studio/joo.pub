验证方式

Luosimao使用HTTP Basic Auth方式进行身份验证，使用api作为验证用户名，API key是验证码，在 管理中心>短信服务>触发发送 查看.

（请注意验证密码需在后台提取的API key之前加上key－ 请仔细查看下列示例）
