接口说明
上行收取

如果客户需要接收用户上行，在相关的申请成功后，可以按照下面的说明进行编程接收,短信接口用户，需要自己做好上行接收数据的接口
提供上行URL（例如：http://XXX.XXX.XXX.XXX/Up.asp ）以便我们给予绑定。

页面推送参数
类型 命名 描述
参数 MsgId 消息ID
参数 MsgID MessageID(短信提交成功后会生成MsgID)
参数 Up_YourNum 自己的特服号号码接口
参数 Up_UserTel 客户上行的手机号码接口
参数 Up_UserMsg 客户上行的短信内容接口

说明：
用户自行写进自己的数据库，以便接收数据。数据接收后请先缓存数据，否则将导致状态报告接收被降权
